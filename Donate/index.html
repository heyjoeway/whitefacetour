Donate

If you’d like to support the hosting/development of this site, you can help support it with a donation. The Whiteface Tour is handled entirely by myself (as an individual, not a business) with no outside funding, and is intended to be non-profit. There are currently no advertisements, although this may change if it puts the hosting of this site in jeopardy. Funds go directly to me, though top priority on the recieved funds goes towards server costs.

You can choose to donate whichever amount you’d like. I use PayPal for accepting donations, although if you’d like to send funds in a different way you can get in touch with me to figure things out.

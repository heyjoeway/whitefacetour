Info

The Whiteface Tour is a project designed by Joseph Judge in cooperation with the Whiteface Atmospheric Sciences Research Center. The goal was to create a virtual tour of the Whiteface summit that can be used as an outreach/educational tool to allow people to simulate exploring the summit with nothing but a computer or mobile phone.

The tour consists of 95 360° images connected together with arrow markers that allow for easy navigation. This tour also works with WebVR/WebXR, allowing VR headsets such as Google Cardboard to be used.
